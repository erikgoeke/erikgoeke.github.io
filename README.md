# Erik Goeke — Mission Profile

Personal site: an immersive scroll journey from Earth orbit to deep space and back, built with Three.js.

## Add your photos

Drop these into `assets/photos/` (the site auto-hides any slot whose file is missing):

| File | What it is |
|---|---|
| `headshot.jpg` | Your profile headshot |
| `jetbot.jpg` | The ML-Bots Jetson robot (save from your LinkedIn post) |
| `students.jpg` | Kids programming robots on the moon-mat (ML-Bots outreach) |
| `classroom.jpg` | You presenting an ML-Bots session at the whiteboard |
| `capitol-hill.jpg` | Group photo in front of the U.S. Capitol |
| `poster.jpg` | You presenting the AE 3610 structures poster |

Tip: in this chat, right-click the `index.html` file card → reveal it in Finder, then copy your photos into `assets/photos/` with the names above.

## Deploy to GitHub Pages

1. Create a repo named `<your-username>.github.io` (e.g. `erikgoeke.github.io`).
2. Push this folder's contents to the repo root (`index.html` at top level).
3. Repo Settings → Pages → Source: **Deploy from a branch** → `main` / root.
4. Your site is live at `https://<your-username>.github.io` within a minute or two.

## Preview locally

Open `index.html` directly in a browser, or run `python3 -m http.server` in this folder and visit http://localhost:8000.
